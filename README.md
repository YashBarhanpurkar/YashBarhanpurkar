### [Automated Optical Quality Inspection System](https://github.com/YashBarhanpurkar/cookie-inspection)
Computer vision pipeline simulating an Industrie 4.0 quality control station. Uses Otsu thresholding + HSV color segmentation to isolate products from conveyor background, GLCM texture analysis (Contrast, Homogeneity, Entropy) for product classification, and pixel-ratio quantification for ingredient distribution вҖ” triggering automatic Accept/Reject decisions against configurable QA thresholds.

`Python` `OpenCV` `scikit-image` `NumPy` `GLCM` `Otsu thresholding` `HSV`

---

### [Industrial Failure Diagnostics & Condition Monitoring](https://github.com/YashBarhanpurkar/Industrial_Failure_Diagnostics_and_Condition_Monitoring)
Diagnostic analytics on 10,000 sensor records (AI4I 2020 dataset). Key finding: thermal data shows only 0.3K difference between healthy and failed machines вҖ” torque is the dominant failure driver, 26% higher in failed units. Random Forest classifier (200 trees) with GridSearchCV 5-fold CV, F1-optimized to balance false alarm rate vs failure capture rate.

`Python` `Scikit-learn` `Random Forest` `GridSearchCV` `Pandas` `Seaborn`
